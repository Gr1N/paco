Installation
============

PyPI
----

You can install the last stable release from PyPI using pip::

    $ pip install paco

GitHub
------

Or install the latest sources from Github::

    $ pip install -e git+git://github.com/h2non/paco.git#egg=

Also you can donwload a source code package from `Github <https://github.com/h2non/paco/tags>`_ and install it using setuptools::

    $ tar xvf -{version}.tar.gz
    $ cd
    $ python setup.py install
